Crystal Magic Mod

A mod of magic and discovery!

Mod is still in alpha, please let me know if you run into any bugs or how you'd like to see the mod improve.

Requirements:

Requires Forge 1.18.2-40.2.13

Installation:

Backup your .minecraft folder
Instal Forge 40.2.13
Run minecraft once and exit.
Add the .jar to your Mods folder
